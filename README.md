# Interactive Advent Calendar

This Advent calendar is built with Bootstrap and JavaScript, providing an exciting and interactive way to count down the days until Christmas. Each day reveals a new surprise, offering festive content, illustrations, or activities to enjoy.

## Features
- **Responsive Design**: Built with Bootstrap, ensuring it works on any device.
- **JavaScript Interactivity**: Engage with the calendar for daily surprises.
- **Customizable**: Easily change the contents behind each calendar door.

## How to Use
1. Clone the repository.
2. Open the `index.html` file in your browser.
3. Enjoy the countdown to Christmas!
